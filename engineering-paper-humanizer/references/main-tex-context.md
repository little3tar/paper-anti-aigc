# 宿主文档背景知识（main.tex 地图）

项目：《硬岩振荡截割装备控制系统设计》

## 第一章：研究背景与意义

- **核心对象**：深部地层，夹矸煤层，高硬度岩石（单轴抗压强度 UCS > 100MPa，坚固系数 $f>10$）
- **核心痛点**：传统悬臂式镐型截齿在此工况下磨损极快，粉尘大，存在火花灾害隐患

## 第二章：国内外研究现状及发展动态

### §2.1 硬岩连续截割技术与MC51装备研发历程
- 钻爆法循环效率瓶颈 vs 机械截割刀具寿命短板，深部硬岩两条路线都不好走
- CSIRO/CMTE 项目 Hood & Alehossein 提出振荡截割（ODC）构想：利用岩石抗拉强度仅为抗压的 1/10~1/20，高频小振幅轴向振荡+底切迫使岩石沿拉伸面破裂，截割反力降低 90%+
- Joy/Komatsu MC51 工程化：碟盘刀具高频冲击逐层剥离，适用 UCS ≤ 250 MPa，平均比能耗 ~3.83 MJ/m³，瞬时截割率 77.01 m³/h
- Erarslan DCD 概念：振荡频率 20~45 Hz 时微裂纹快速扩展，岩屑体积增加 ~200%，法向力降低 25%~30%，比能耗最高降低 45%
- **关键警示**：梁明纯粉砂岩实验——循环上限应力低于疲劳门槛时压密强化反而使强度升高 8.5%，振荡参数必须超过疲劳门槛

### §2.2 振荡截割工艺特征与装备工作原理
- 碟盘刀具运动学：瞬时切深 δ(t) = vt + e[1-cos(ωt)]，eω≫v 时振荡主导
- Kovalyshen 解析模型：ODC 降低反力但不减少总功率，能量在振荡与进给机构间重新分配
- 刘春生：振动截割时楔面能耗小于无振动，破岩以拉剪耦合为主；齿数与径向载荷呈下凹抛物线关系
- MC51 自动截割逻辑（Jokonya 专利）：推进油缸压力实时监测感知煤岩界面，两条硬约束——逆运动学毫秒级解算、压力反馈带宽须覆盖 20~50 Hz

### §2.3 少自由度并联机构运动学与动力学
- 3-RPS 逆解：矢量闭环 L_i = P + R_rot·b_i - B_i，取模长即为液压缸目标位移
- CGA 方法（赵福琛）：坐标系无关建模，对称 3-RPS 正逆解均有解析解
- 单彦霞：对偶四元数 D-H + 虚拟支链法，半角正切转多项式，Maple/MATLAB GUI 自动求解
- 动力学：张海峰分解建模（拉格朗日+虚功原理），刘英建刚柔耦合单元改善位姿误差

### §2.4 电液伺服系统与运动控制
- 固有非线性：流量-压力耦合、油液可压缩性、密封摩擦、阀口死区
- Bessa 自适应模糊滑模：滑模面 s=ce+ė，模糊逻辑在线逼近死区不确定项
- Dai 前馈-反馈复合抗扰 + 死区逆模型补偿
- 宋启琛：卡尔曼滤波 + 模糊 PID，跟踪误差降低 18.94%，有效频宽 9 Hz
- 并联层面：李信言多参数滑模解耦，张李诚 SDC 法向力误差降低 72.6%
- 数据驱动：武建新 SAC 强化学习自动调 PID，陆振宇 DRL 跨系统迁移（方波静态误差 ≤0.35 mm）
- 韩丁：并联机构逆解 + ESO-LTVMPC 电液伺服位移跟踪，架构与本课题截割臂控制相似

### §2.5 本章小结
- 振荡截割降反力/比能耗有充分支撑，MC51 验证技术路线可行
- 3-RPS 运动学正反解方法趋于成熟，动力学与刚度分析有阶段性进展
- 电液伺服从经典滑模发展到融合模糊/强化学习的智能方法
- **不足**：将并联构型设计、运动学、动力学与电液伺服控制贯通为面向硬岩截割的完整系统方案，仍然少见

## 第三章：初步设计方案

### §3.1 总体方案与功率流向匹配设计
- 整机装机功率 320 kW，净切削功率 $P_{cut} \approx 81.9$ kW（SE=3.83 MJ/m³ × ICR=77.01 m³/h，公式 eq:power_calc）
- 功率流向三条损耗路径：液压系统综合损耗（容积泄漏+节流压降+沿程摩阻，系统总效率 ≤ 0.85）、振荡驱动内部损耗（20~50 Hz 偏心机构摩擦+高频换向冲击）、辅助作业机构能耗（履带行走+除尘冷却+物料输送）
- 总装机功率与净切削功率接近 4:1，为能量传递链条物理损耗留出工程冗余

### §3.2 截割机构（机械）方案设计与自由度计算
- 放弃串联大悬臂，采用 **3-RPS 并联机构** 承担空间位姿调整（切深进给+切入角偏转）
- 截割头自身旋转和高频轴向振荡由内置液压马达与偏心轴机构独立驱动，不经并联支链传递
- 三条电液伺服驱动链非对称协调伸缩，低速进给与高频破岩物理解耦
- **自由度计算**：Kutzbach 公式，N=8, j=9, Σfi=15, F=6×(8-9-1)+15=3
- 3 DOF = Z 轴平移（切深进给）+ X 轴俯仰 + Y 轴横滚 → 2R1T 构型
- 逆解矢量方程：$L_i = P + R_{rot} \cdot b_i - B_i$，取模长即液压缸目标位移 $L_{ref}$（3.2 节到此结束，功率计算已前置至 §3.1）

### §3.3 液压驱动与控制方案设计及可行性论证
- 负载敏感（LS）闭心电液比例控制系统方案选型论证：多缸联动工况下节流损失与溢流发热问题，LS 系统原理（梭阀网络采集最高负载压力，泵输出压力 = 最高负载 + ΔP_LS）
- LS 系统原理拓扑图（fig:ls_principle，TikZ 绘制）
- 截割载荷映射与执行元件受力分析原理：力雅可比矩阵 $\boldsymbol{J}^T$ 将末端力旋量映射为三缸轴向力（eq:force_jacobian）
- 明确缸径、排量等具体参数计算推迟到第四章详细设计阶段

### §3.4 控制系统软硬件总体架构方案
- 上位机（工控机/触摸屏）+ 下位机（西门子 S7-1500F PLC）分布式硬件架构
- PLC 选型依据：30 ns 位运算、2 ms 内完成 3-RPS 逆运动学解算、故障安全型（F 系列）满足井下防爆要求
- 感知端：MTS R 系列磁致伸缩位移传感器（SSI 总线，分辨率 5 μm）+ 压力变送器（4~20 mA）
- I/O 信号地址分配表（tab:io_map），含 A 缸无杆腔压力通道 IW 66
- 核心控制算法：外层运动学解算 + 内层位置-压力双闭环复合控制
- 增量式 PID（eq:pid）+ 死区补偿函数（eq:deadzone）+ 压力前馈补偿
- 控制系统框图（fig:control_block，TikZ 绘制，含压力反馈虚线回路）

### §3.5 本章小结
- 回顾全章三层面（机械本体、液压驱动、控制算法）初步方案设计
- 机械：3-RPS 并联机构替代串联悬臂，低频空间进给与高频振荡破岩物理解耦
- 液压：负载敏感闭心系统消除多余溢流损耗，适应三缸负载差异大、变化快的工况
- 控制：位置-压力复合双闭环 + 死区逆向补偿 + 压力前馈，抑制载荷突变对轨迹的扰动
- 功率匹配：净切削功率 81.9 kW / 装机 320 kW ≈ 1:4，雅可比矩阵力学映射替代经验预设
