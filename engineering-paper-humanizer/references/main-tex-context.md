# 宿主文档背景知识（main.tex 地图）

项目：《硬岩振荡截割装备控制系统设计》

## 第一章：研究背景与意义

- **核心对象**：深部地层，夹矸煤层，高硬度岩石（单轴抗压强度 UCS > 100MPa，坚固系数 $f>10$）
- **核心痛点**：传统悬臂式镐型截齿在此工况下磨损极快，粉尘大，存在火花灾害隐患

## 第二章：国内外研究现状

### §2.1 硬岩连续截割技术与MC51装备研发历程
- 钻爆法循环效率瓶颈 vs 机械截割刀具寿命短板，深部硬岩两条路线都不好走
- CSIRO/CMTE 项目 Hood & Alehossein 提出振荡截割（ODC）构想：利用岩石抗拉强度仅为抗压的 1/10~1/20，高频小振幅轴向振荡+底切迫使岩石沿拉伸面破裂，截割反力降低 90%+
- Joy/Komatsu MC51 工程化：碟盘刀具高频冲击逐层剥离，适用 UCS ≤ 250 MPa，平均比能耗 ~3.83 MJ/m³，瞬时截割率 77.01 m³/h
- Erarslan DCD 概念：振荡频率 20~45 Hz 时微裂纹快速扩展，岩屑体积增加 ~200%，法向力降低 25%~30%，比能耗最高降低 45%
- **关键警示**：梁明纯粉砂岩实验——循环上限应力低于疲劳门槛时压密强化反而使强度升高 8.5%，振荡参数必须超过疲劳门槛

### §2.2 振荡截割工艺特征与装备工作原理
- 碟盘刀具运动学：瞬时切深 δ(t) = vt + e[1-cos(ωt)]，eω≫v 时振荡主导
- Kovalyshen 解析模型：ODC 降低反力但不减少总功率，能量在振荡与进给机构间重新分配
- 刘春生：振动截割时楔面能耗小于无振动，破岩以拉剪耦合为主；齿数与径向载荷呈下凹抛物线关系
- MC51 自动截割逻辑（Jokonya 专利）：推进油缸压力实时监测感知煤岩界面，两条硬约束——逆运动学毫秒级解算、压力反馈带宽须覆盖 20~50 Hz

### §2.3 少自由度并联机构运动学与动力学
- 3-RPS 逆解：矢量闭环 L_i = P + R_rot·b_i - B_i，取模长即为液压缸目标位移
- CGA 方法（赵福琛）：坐标系无关建模，对称 3-RPS 正逆解均有解析解
- 单彦霞：对偶四元数 D-H + 虚拟支链法，半角正切转多项式，Maple/MATLAB GUI 自动求解
- 动力学：张海峰分解建模（拉格朗日+虚功原理），刘英建刚柔耦合单元改善位姿误差

### §2.4 电液伺服系统与运动控制
- 固有非线性：流量-压力耦合、油液可压缩性、密封摩擦、阀口死区
- Bessa 自适应模糊滑模：滑模面 s=ce+ė，模糊逻辑在线逼近死区不确定项
- Dai 前馈-反馈复合抗扰 + 死区逆模型补偿
- 宋启琛：卡尔曼滤波 + 模糊 PID，跟踪误差降低 18.94%，有效频宽 9 Hz
- 并联层面：李信言多参数滑模解耦，张李诚 SDC 法向力误差降低 72.6%
- 数据驱动：武建新 SAC 强化学习自动调 PID，陆振宇 DRL 跨系统迁移（方波静态误差 ≤0.35 mm）
- 韩丁：并联机构逆解 + ESO-LTVMPC 电液伺服位移跟踪，架构与本课题截割臂控制相似

### §2.5 本章小结
- 振荡截割降反力/比能耗有充分支撑，MC51 验证技术路线可行
- 3-RPS 运动学正反解方法趋于成熟，动力学与刚度分析有阶段性进展
- 电液伺服从经典滑模发展到融合模糊/强化学习的智能方法
- **不足**：将并联构型设计、运动学、动力学与电液伺服控制贯通为面向硬岩截割的完整系统方案，仍然少见

## 第三章：初步设计方案

### 截割机构（机械层）

弃用串联大悬臂，引入兼顾刚度与位姿微调的 **3-RPS 并联机构**（多支链支撑，抗让刀刚度高）。配合前端高频振荡驱动（防压剪，主攻拉剪）。

### 液压系统（驱动层）

系统耐压基准 $P_{sys}=31.5\text{ MPa}$。采用 **负载敏感 (LS) 系统**（结合蓄能器），避免重载工况的节流发热。

### 控制系统（电气与算法层）

- 西门子 S7-1500F 故障安全型 PLC 为控核
- MTS 系列磁致伸缩位移传感器（SSI总线）
- 单缸算法：**增量式 PID 控制叠加非线性死区补偿律**（比例阀芯存在掩盖区）
