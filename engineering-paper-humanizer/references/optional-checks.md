# 可选检查与质量评分

> 本文件包含 engineering-paper-humanizer 技能的可选检查内容，仅在用户明确要求时使用。

## 质量评分标准（可选自检）

对改写后的文本进行 1-10 分评估（总分 50）：

| 维度       | 评估标准                                                          | 得分    |
| ---------- | ----------------------------------------------------------------- | ------- |
| **直接性** | 直接陈述事实还是绕圈宣告？（10分：直截了当；1分：充满铺垫）       | /10     |
| **节奏**   | 句子长度是否变化，有无突发性？（10分：长短交错；1分：机械重复）   | /10     |
| **信任度** | 是否尊重读者智慧，有无过度解释？（10分：简洁明了；1分：过度解释） | /10     |
| **真实性** | 有无工程妥协感和具体参数？（10分：硬核真实；1分：空泛完美）       | /10     |
| **精炼度** | 还有可删减的填充词吗？（10分：无冗余；1分：大量废话）             | /10     |
| **总分**   |                                                                   | **/50** |

**评分标准：**

- 45-50 分：优秀，已彻底去除 AI 痕迹并注入工程灵魂
- 35-44 分：良好，仍有改进空间
- 低于 35 分：需要重新修订

## 使用说明

当用户明确要求评估 AI 浓度或进行质量评分时，按此标准执行评估。评估结果可作为改写质量的参考，但不影响核心改写流程。
