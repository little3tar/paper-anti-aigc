# AIGC 痕迹必杀：七大核心维度

## 维度一：降解"宏大叙事"与"空泛表达"

真实的工程师行文客观克制，绝不使用口号式的虚词填补字数。

- **砍掉起手式废话**：坚决删除"为了能够有效解决...重大难题"、"本课题主要目的是..."、"应运而生"。直接抛出技术动作或工况事实。
- **降级绝对化词汇**：将"彻底解决"、"卓越性能"降级为平实的"有效改善"、"显著降低"、"满足...需求"。
- **去除空泛呼吁**：禁用"具有十分重要的工程应用价值"。改为具体学术贡献，例如："为深部硬岩巷道的智能化掘进奠定控制系统基础"。
- **消解"伪深度"演说风**：斩断试图将技术细节（如比例阀死区补偿算法或单腔面积推演）强行拔高至抽象层面的句子，回归硬核物理论述。

## 维度二：打破"机械结构"与"流水线排版"

AI 极度痴迷于对称和预告，必须彻底打碎这种规律性。

- **抹除章节预告**：禁止在段首写"本章将针对...开展工作"或"接下来将详细阐述..."。运用客观主语自然过渡。
- **消灭八股列表**：禁止使用"名词+冒号+长篇解释"的清单体。将 `enumerate/itemize` 碎片化内容融合成长短不一、详略得当的连贯段落。
- **打破看图说话**：禁止在插入图片后紧跟独立解释段。图片引用自然融入主正文（如："该动作在物理架构上实现了降维解耦（见图 2）。"）。
- **避免对称排比**：严禁结构完全对等的"三段式"排比句，以及滥用"不仅...反而..."的机械平行结构。

## 维度三：重塑"僵化逻辑"与"过渡句式"

用真实工程的物理因果关系推动行文。

- **根除"虽然...但是"模板**：直接客观陈述双方侧重点。
    - 示例："目前国内研究主要侧重于碟盘刀具破岩机理与动力学仿真，而针对硬岩振荡截割整机的工程化设计尚处于起步阶段。"
- **克制连接词泛滥**：强制删除至少 50% 的强连接词，用工况条件推导天然结论（如：瞬时峰值大 → 被迫选用大缸径）。
- **删除废话跳板**：删除"这种传统模式存在许多问题"等无信息增量的过渡句，直击痛点。

## 维度四：修复"标点暴露"与"LaTeX格式漏洞"

- **引用位置红线**：`\cite{...}` 紧贴被引文字，禁止前加空格，标记在句号/逗号内侧。
    - ❌ `控制任务。 \cite{xxx}`
    - ✅ `控制任务\cite{xxx}。`
- **引号规范**：强制中文双引号 " "。
- **括号清理**：避免破折号做长解释，严禁堆砌连续括号，解释性文字直接化入正文。

## 维度六：重构"文献综述"起手式

- ❌ "现有研究表明..." → ✅ "某学者指出..."或直接陈述结论
- ❌ "国内起步较晚..." → ✅ 删去"起步晚"，直接陈述"近年来，国内学者围绕XXX展开了针对性研究"
- ❌ "...这为控制闭环反馈提供了数据支撑"（升华结尾）→ ✅ 删去，陈述完技术事实即结束自然段

## 维度七：提振突发性（Burstiness）与扰动困惑度（Perplexity）

主流 AI 检测算法通过扫描困惑度和突发性判断文本来源。

- **提振突发性**：主动使用 3~5 字极简短句（"传统模式难以为继。"），紧接 20+ 字参数密集长句（"基于帕斯卡原理核算，为对抗逾 100kN 的瞬时反向冲击，单缸无杆腔理论面积被设定为不小于 0.0167 m²。"）。人为制造大开大合"顿挫感"。
- **扰动困惑度**：刻意保留工程粗糙感词汇（"受限于井下防爆空间定额，妥协保留了..."、"实测结果略有出入"、"被迫采用"）。关键工程细节处保留人类思维的"毛刺"。

## 快速速查：Anti-Patterns

| 现象                               | 判定         | 处理                           |
| ---------------------------------- | ------------ | ------------------------------ |
| 段首出现"本章将针对...展开工作"    | 章节预告     | 直接删除，用客观主语过渡       |
| "名词+冒号+长篇解释"列表体         | 八股列表     | 融合成长短不一的连贯段落       |
| "虽然取得了显著进展，但仍存在短板" | AI 模板句    | 直接客观陈述双方侧重点         |
| 每段开头都是"因此/然而/值得注意"   | 连接词泛滥   | 强制删除 ≥ 50%，用物理因果推导 |
| 所有句子长度 15~20 字且节奏均匀    | 低突发性     | 插入极短句和超长参数句         |
| "突破技术瓶颈""提供数据支撑"       | AI 敏感词    | 按降重字典替换                 |
| `\cite{}` 前有空格或在句号外       | 引用格式错   | 修复为紧贴引文、句号内侧       |
| 修改了 `\equation` 内部公式        | 破坏数学环境 | 🚫 绝对禁止                    |
| 出现文档中没有的数值               | 捏造数据     | 🚫 绝对禁止                    |
