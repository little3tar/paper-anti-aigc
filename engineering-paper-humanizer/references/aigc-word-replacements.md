# AI 高频敏感词组降重替换字典

## 困境 / 意图类

| ❌ AI 高频词  | ✅ 替换为              |
| ------------- | ---------------------- |
| 双重困境      | 双重问题、局限性       |
| 协同难题      | 协同障碍、控制挑战     |
| 旨在解决/探讨 | 致力于改善、深入分析了 |

## 突破 / 价值支撑类

| ❌ AI 高频词      | ✅ 替换为                                                  |
| ----------------- | ---------------------------------------------------------- |
| 突破...技术瓶颈   | 克服...技术难点                                            |
| 取得了一定进展    | 直接删除或替换为具体科研动作（如"提出了基于小波包的算法"） |
| 提供理论/数据支撑 | 直接删除或改写为"为...奠定基础"                            |

## 高频啰嗦副词

| ❌ AI 高频词               | ✅ 替换为              |
| -------------------------- | ---------------------- |
| 有效（有效突破、有效诱导） | 直接删除               |
| 从而实现 / 以期            | 以保证、迫使、以实现   |
| 进一步加剧了这一状况       | 使...进一步恶化        |
| 应运而生                   | 直接删除               |
| 具有十分重要的工程应用价值 | 直接删除或改为具体贡献 |

## 新增分类一：通用 AI 高频词（来自 Humanizer-zh）

| ❌ AI 高频词         | ✅ 替换为                    |
| -------------------- | ---------------------------- |
| 此外                 | 直接删除，或改为具体逻辑连接 |
| 至关重要             | 重要、关键（视语境选用）     |
| 深入探讨             | 分析、研究、讨论             |
| 强调（动词滥用）     | 指出、表明、提出             |
| 持久的               | 长期的、稳定的               |
| 增强                 | 提升、改善、加强             |
| 培养                 | 直接删除或改为具体动作       |
| 获得（抽象用法）     | 取得、得到                   |
| 突出（动词）         | 体现、反映                   |
| 相互作用             | 耦合、交互、影响             |
| 复杂/复杂性          | 直接删除或用具体技术描述替代 |
| 格局（抽象名词）     | 直接删除或改为具体领域       |
| 关键性的             | 关键的、重要的               |
| 展示                 | 表明、显示、呈现             |
| 织锦（抽象名词）     | 直接删除                     |
| 证明（抽象升华用法） | 表明、验证                   |
| 宝贵的               | 重要的、有价值的             |
| 充满活力的           | 直接删除                     |
| 不断演变的           | 直接删除或改为具体变化描述   |
| 不可磨灭的           | 直接删除                     |

## 新增分类二：填充短语降重表

| ❌ 填充短语      | ✅ 替换为            |
| ---------------- | -------------------- |
| 为了实现这一目标 | 为此                 |
| 由于……的事实     | 因为                 |
| 在这个时间点     | 目前                 |
| 在……的情况下     | 若/当                |
| 具有……的能力     | 能够/可以            |
| 值得注意的是     | 直接删除，直述事实   |
| 显而易见         | 直接删除             |
| 不可否认         | 直接删除             |
| 综上所述         | 视语境决定保留或删除 |

## 新增分类三：模糊归因降重表

| ❌ 模糊归因    | ✅ 替换为                            |
| -------------- | ------------------------------------ |
| 行业报告显示   | 引用具体文献（\cite{xxx}）           |
| 观察者指出     | 引用具体学者或文献                   |
| 专家认为       | 引用具体学者姓名+文献                |
| 一些批评者认为 | 引用具体文献或删除                   |
| 现有研究表明   | 具体引用（如"张某等\cite{xxx}指出"） |
| 研究人员发现   | 具体引用文献                         |

## 新增分类四：宣传/广告语言降重表

| ❌ 宣传语言       | ✅ 替换为                  |
| ----------------- | -------------------------- |
| 开创性的          | 新型的、改进的             |
| 令人叹为观止的    | 直接删除                   |
| 著名的            | 直接删除或改为"广泛应用的" |
| 卓越性能          | 良好性能、满足要求         |
| 无缝（衔接/体验） | 直接删除                   |
| 充满活力的        | 直接删除                   |
| 坐落于            | 位于                       |
