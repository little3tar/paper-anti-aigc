# AI 高频敏感词组降重替换字典

## 困境 / 意图类

| ❌ AI 高频词  | ✅ 替换为              |
| ------------- | ---------------------- |
| 双重困境      | 双重问题、局限性       |
| 协同难题      | 协同障碍、控制挑战     |
| 旨在解决/探讨 | 致力于改善、深入分析了 |

## 突破 / 价值支撑类

| ❌ AI 高频词      | ✅ 替换为                                                  |
| ----------------- | ---------------------------------------------------------- |
| 突破...技术瓶颈   | 克服...技术难点                                            |
| 取得了一定进展    | 直接删除或替换为具体科研动作（如"提出了基于小波包的算法"） |
| 提供理论/数据支撑 | 直接删除或改写为"为...奠定基础"                            |

## 高频啰嗦副词

| ❌ AI 高频词               | ✅ 替换为              |
| -------------------------- | ---------------------- |
| 有效（有效突破、有效诱导） | 直接删除               |
| 从而实现 / 以期            | 以保证、迫使、以实现   |
| 进一步加剧了这一状况       | 使...进一步恶化        |
| 应运而生                   | 直接删除               |
| 具有十分重要的工程应用价值 | 直接删除或改为具体贡献 |
